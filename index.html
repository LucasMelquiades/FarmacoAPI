Sobre este
projeto

Este projeto nasceu da curiosidade e do desejo de organizar dados relevantes para diferentes comunidades interessadas em saúde e fitoterapia. Ao explorar o Formulário de Fitoterápicos da Farmacopeia Brasileira – 2ª edição, percebi seu enorme potencial como ferramenta de democratização do conhecimento técnico, especialmente no campo dos medicamentos tradicionais e naturais.

Com o objetivo de facilitar o acesso e a consulta às formulações descritas na publicação oficial, desenvolvi um banco de dados em formato JSON que estrutura e apresenta as informações de forma dinâmica e acessível por meio deste site.

É importante destacar que este projeto não substitui a publicação original da Farmacopeia, nem pretende oferecer orientação terapêutica ou farmacêutica. Trata-se de uma iniciativa informativa e organizacional, voltada à valorização do conteúdo técnico já publicado.

Recomenda-se fortemente que os usuários consultem o Formulário de Fitoterápicos da Farmacopeia Brasileira – 2ª edição para obter detalhes completos sobre as formulações, referências bibliográficas e demais informações regulamentares.

Aviso de Responsabilidade: Este site tem caráter exclusivamente informativo. O autor não se responsabiliza pelo uso indevido das formulações aqui apresentadas. Todas as informações são baseadas na publicação oficial do Formulário de Fitoterápicos da Farmacopeia Brasileira – 2ª edição, cuja consulta é indispensável para qualquer aplicação prática.

As imagens do site são trazidas automaticamente via API do Wikimedia Commons. Os créditos e as licenças estão sempre visíveis abaixo de cada imagem, respeitando os termos de uso.
